# YouTube Automation Script

Автоматизация действий в YouTube с записью видео с веб-камеры.

## 🎯 Что делает

1. Открывает YouTube в браузере
2. Проверяет запуск Chrome
3. Выполняет серию кликов мышью
4. Записывает видео с веб-камеры (5 секунд)
5. Сохраняет видео и открывает папку
